# meme_generator

 Meme generator dynamically generates memes, including an image with an overlaid quote.



## Installation
```
https://github.com/tars13/meme-generator.git
```

### As web app

```
python app.py
```

and open http://127.0.0.1:5000/


## Requirements
* pandas
* flask
* requests
* python-docx
